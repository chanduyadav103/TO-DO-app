## **What are HTML Web APIs?**

**Web APIs (Application Programming Interfaces)** are **JavaScript-accessible interfaces provided by browsers** that allow developers to interact with the **browser and the user’s system**. They provide functionality beyond static HTML, enabling **dynamic web applications**.

Key points:

- Built into the browser — no extra libraries needed.
- Expose capabilities like **DOM manipulation, network requests, storage, audio/video processing, geolocation, notifications, etc.**
- Used to create **interactive, modern web applications**.

---

## **List of Important Web APIs**

| API                                                    | Purpose / Explanation                                                              |
| ------------------------------------------------------ | ---------------------------------------------------------------------------------- |
| **DOM API**                                            | Access and manipulate HTML elements dynamically.                                   |
| **Fetch API**                                          | Make HTTP requests (AJAX) to fetch or send data.                                   |
| **LocalStorage / SessionStorage**                      | Store key-value data in the browser for persistence (local or session-based).      |
| **Geolocation API**                                    | Get the user’s current geographic location.                                        |
| **Canvas API**                                         | Draw 2D graphics and animations using JavaScript.                                  |
| **Web Audio API**                                      | Create, process, and manipulate audio in the browser.                              |
| **Notification API**                                   | Display desktop or mobile notifications to the user.                               |
| **Drag & Drop API**                                    | Enable drag-and-drop interactions with elements.                                   |
| **IntersectionObserver API**                           | Detect when elements enter or leave the viewport (used for lazy loading).          |
| **Web Storage API**                                    | Includes LocalStorage and SessionStorage for storing persistent or temporary data. |
| **Web Workers API**                                    | Run scripts in the background for parallel processing.                             |
| **History API**                                        | Manage browser history and update URLs without refreshing pages (SPA behavior).    |
| **Pointer Events API**                                 | Handle user input from mouse, touch, pen devices in a unified way.                 |
| **Clipboard API**                                      | Read from and write to the system clipboard.                                       |
| **Battery Status API** _(deprecated in most browsers)_ | Access battery status information.                                                 |
| **Device Orientation / Motion API**                    | Detect device orientation or motion (mobile-specific).                             |
| **Push API**                                           | Enable push notifications for web apps.                                            |
| **MediaDevices API**                                   | Access camera, microphone, and other media input devices.                          |
| **Vibration API**                                      | Control device vibration for alerts and feedback (mostly mobile).                  |
| **Payment Request API**                                | Streamline payment workflows in web apps.                                          |
