1. Write a function to add two numbers.
   Create a function add(a, b) that returns the sum of a and b. Call the function with different numbers and print the results.

2. Function to check even or odd.
   Write a function isEven(n) that returns "Even" if the number is even, "Odd" otherwise. Test it with various numbers.

3. Function to find the largest of three numbers.
   Write a function maxOfThree(a, b, c) that returns the largest of the three numbers. Try it with both positive and negative numbers.

4. Function to reverse a string.
   Write a function reverseString(str) that returns the reverse of the input string. Example: reverseString("hello") should return "olleh".

5. Function to calculate factorial.
   Write a function factorial(n) that returns the factorial of a given number. Example: factorial(5) should return 120. Use a loop, not recursion.

6. Function expression to find if a string is a palindrome.
   Create a function assigned to a variable isPalindrome that checks if a string is a palindrome (same forward and backward). Example: isPalindrome("level") should return true.

7. Function to count the number of vowels in a string.
   Write a function countVowels(str) that returns the number of vowels in the input string. Test with different strings.

8. Function to find the sum of all elements in an array.
   Write a function sumArray(arr) that takes an array of numbers and returns their sum. Example: sumArray(\[1, 2, 3, 4]) should return 10.

9. Function to check if a number is prime.
   Write a function isPrime(n) that returns true if n is a prime number, false otherwise. Test with several numbers.

10. Function to remove all spaces from a string.
    Write a function removeSpaces(str) that returns a new string with all spaces removed from the original string. Example: removeSpaces("hello world") should return "helloworld".
11. Write a function that checks if a given number is positive, negative, or zero using if-else-if statements.
    Example: checkNumber(-5) should print "Negative", checkNumber(0) should print "Zero", checkNumber(4) should print "Positive".

12. Write a function that prints all numbers from 1 to n using a for loop.
    Example: printNumbers(5) should print 1 2 3 4 5.

13. Write a function that prints all even numbers between 1 and n using a for loop and if statement.
    Example: printEvenNumbers(10) should print 2 4 6 8 10.

14. Write a function that prints the multiplication table of a given number up to 10 using a for loop.
    Example: printTable(3) should print 3 6 9 12 15 18 21 24 27 30.

15. Write a function that takes a character and determines if it is a vowel or consonant using if-else and switch.
    Example: checkChar('a') should print "Vowel", checkChar('b') should print "Consonant".

16. Write a function that counts and prints the number of positive, negative, and zero values in an array using for loop and if-else-if.
    Example: countValues(\[2, -1, 0, 3, -5, 0]) should print counts for positive, negative, and zero.

17. Write a function that uses a for loop and if-else to find the maximum number in an array.
    Example: findMax(\[2, 8, 1, 5, 9]) should print 9.

18. Write a function that takes a day number (1-7) and prints the corresponding weekday using switch statement.
    Example: printDay(1) should print "Monday", printDay(7) should print "Sunday".

19. Write a function that calculates the sum of all odd numbers from 1 to n using for loop and if.
    Example: sumOddNumbers(10) should return 25 (1 + 3 + 5 + 7 + 9).

20. Write a function that takes an array of numbers and prints "High" for values greater than 100, "Medium" for values between 50 and 100 (inclusive), and "Low" for values less than 50, using for loop and if-else-if.
    Example: labelArray(\[45, 78, 120, 30, 99]) should print Low Medium High Low Medium.
