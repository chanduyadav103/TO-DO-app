1. forEach
Description:
Runs a provided function once for each array element. Doesn’t return anything.


// Example 1: Log each name
const names = ['Alice', 'Bob', 'Carol'];
names.forEach(name => console.log(name));
// Output: Alice, Bob, Carol

// Example 2: Increment DOM counters
const counters = document.querySelectorAll('.counter');
counters.forEach((el, i) => el.textContent = `#${i + 1}`);
Real-world usage:
Sending a notification to each user in a list, or updating a list of UI elements one by one.

2. map
Description:
Transforms every element in an array and returns a new array of the same length.


// Example 1: Square each number
const nums = [1, 2, 3];
const squares = nums.map(n => n * n);
// squares → [1, 4, 9]

// Example 2: Extract names from objects
const users = [{name:'Alice',id:1},{name:'Bob',id:2}];
const names = users.map(u => u.name);
// names → ['Alice', 'Bob']
Real-world usage:
Converting an array of product objects into an array of just their prices for charting.

3. reduce
Scenario: Calculating the total cost in a shopping cart before checkout.


const cart = [
  { name: 'T-Shirt', price: 15 },
  { name: 'Jeans',   price: 40 },
  { name: 'Socks',   price:  5 },
];
const total = cart.reduce((sum, item) => sum + item.price, 0);
console.log(total); // 60
Real-time use: On your e-commerce site’s cart page, run this when the user clicks “View Total” to show “Your total is $60.”

filter
Description:
Returns a new array containing only elements that pass a given test.


// Example 1: Only even numbers
const nums = [1,2,3,4];
const evens = nums.filter(n => n % 2 === 0);
// evens → [2, 4]

// Example 2: Active users
const users = [{active:true},{active:false}];
const activeUsers = users.filter(u => u.active);
// activeUsers → [{active:true}]
Real-world usage:
Showing only “in-stock” products on a page, or filtering search results by user-selected criteria.


some
Scenario: Checking if any event tickets are sold out.


const events = [
  { name: 'Concert A', seatsLeft:  0 },
  { name: 'Concert B', seatsLeft: 10 },
];
const anySoldOut = events.some(e => e.seatsLeft === 0);
console.log(anySoldOut); // true
Real-time use: Before rendering the “Buy Tickets” button, disable it or show “Sold Out” if anySoldOut is true.

every
Scenario: Validating that all quiz questions have been answered.


const answers = ['A', 'B', '', 'D'];
const allAnswered = answers.every(ans => ans !== '');
console.log(allAnswered); // false
Real-time use: On your quiz app, only enable the “Submit” button when allAnswered is true, preventing incomplete submissions.

find
Description:
Returns the first element that passes a test; otherwise undefined.


// Example 1: First even number
const nums = [1, 3, 4, 6];
nums.find(n => n % 2 === 0);
// → 4

// Example 2: Find user by ID
const users = [{id:1},{id:2}];
users.find(u => u.id === 2);
// → {id: 2}
Real-world usage:
Looking up an order by its ID in an array of orders.

entries
Description:
Returns an iterator of [index, value] pairs.


// Example 1: Log index + value
const letters = ['a','b','c'];
for (const [i, l] of letters.entries()) {
  console.log(i, l);
}
// → 0 'a', 1 'b', 2 'c'

// Example 2: Convert to object
const arr = ['x','y'];
const obj = Object.fromEntries(arr.entries());
// obj → { '0':'x', '1':'y' }
Real-world usage:
Generating numbered list items in a UI, or converting arrays into objects for lookup tables.

flat
Description:
Flattens nested arrays by one level (by default).


// Example 1: One-level flatten
const arr = [1, [2, 3], 4];
arr.flat();
// → [1, 2, 3, 4]

// Example 2: Deeper flatten
const deep = [1, [2, [3]]];
deep.flat(2);
// → [1, 2, 3]
Real-world usage:
Flattening nested comment replies into a single list for display.

flatMap
Scenario: Aggregating all tags from a blog’s posts into one flat list.


const posts = [
  { title: 'JS Tricks', tags: ['js','tips'] },
  { title: 'CSS Grid', tags: ['css','layout'] },
];
const allTags = posts.flatMap(p => p.tags);
console.log(allTags); // ['js','tips','css','layout']
Real-time use: Build a “Tag Cloud” sidebar by flattening each post’s tags so readers can click any tag to filter posts.
fill
Description:
Fills all (or a portion of) an array with a static value.


// Example 1: Fill entire array
const arr = new Array(4).fill(0);
// → [0,0,0,0]

// Example 2: Partial fill
const nums = [1,2,3,4];
nums.fill('x', 1, 3);
// → [1,'x','x',4]
Real-world usage:
Initializing a matrix or placeholder values before real data loads.

sort
Description:
Sorts the array in-place—by default lexicographically, but you can provide a comparator.


// Example 1: Numeric sort ascending
const nums = [10, 2, 30];
nums.sort((a, b) => a - b);
// → [2, 10, 30]

// Example 2: Alphabetical sort
const fruits = ['banana','apple','cherry'];
fruits.sort();
// → ['apple','banana','cherry']
Real-world usage:
Ordering products by price or name in a shopping app.

slice
Description:
Returns a shallow copy of a portion of the array (start inclusive, end exclusive).


// Example 1: First 2 items
const arr = [1,2,3,4];
arr.slice(0,2);
// → [1,2]

// Example 2: Last 2 items
arr.slice(-2);
// → [3,4]
Real-world usage:
Paginating search results (e.g., showing items 10–19 from a large list).

shift
Description:
Removes and returns the first element. Mutates the array.


// Example 1: Queue pop
const queue = ['a','b','c'];
const first = queue.shift();
// first → 'a', queue → ['b','c']

// Example 2: Remove header from CSV row
const row = ['Name','Age','City'];
row.shift();
// row → ['Age','City']
Real-world usage:
Processing the next task from a task queue in order of arrival.

unshift
Description:
Adds one or more elements to the beginning and returns the new length.


// Example 1: Add priority task
const tasks = ['task2','task3'];
tasks.unshift('task1');
// tasks → ['task1','task2','task3']

// Example 2: Prepend a signature line
const lines = ['Line A','Line B'];
lines.unshift('=== START ===');
// lines → ['=== START ===','Line A','Line B']
Real-world usage:
Prepending a “Most Recent” marker to a list of messages.

splice
Scenario: Updating a playlist when a user removes one song and adds two new ones in its place.


let playlist = ['Song A', 'Song B', 'Song C', 'Song D'];
// At index 1, remove 1 song and insert 'Song X' and 'Song Y':
playlist.splice(1, 1, 'Song X', 'Song Y');
console.log(playlist);
// → ['Song A', 'Song X', 'Song Y', 'Song C', 'Song D']
Real-time use: In a music app, reflect the user’s action immediately: remove the old track and show the two new ones seamlessly in their playlist view.
