Stock price prediction using deep learning

Stock price prediction is a crucial task in financial markets, where accurate forecasting can aid investors in making informed decisions. Stock price prediction using deep learning which is a subset of machine learning Uses techniques such as LSTM networks. This project leverages Long Short-Term Memory (LSTM) neural networks to predict future stock prices based on historical stock data. Utilizing Yahoo Finance the model fetches real-time stock prices and preprocesses the data using Minmax scaling for normalization. Key features such as closing price and trading volume are used to train the LSTM model.
